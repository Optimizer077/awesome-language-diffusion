# Contributing to Awesome Language Diffusion

Thanks for helping keep this list accurate and useful! Contributions of new
papers, models, code, corrections, and links are welcome.

## How to add an entry

1. Fork the repo and create a branch.
2. Add your entry to the most relevant section in `README.md` (or a file under `topics/`).
3. Use the standard row format:

   ```
   | **Title** | Venue'YY | [abs](https://arxiv.org/abs/XXXX.XXXXX) · [code](URL) | One-line why-it-matters. |
   ```

4. Keep the one-line description factual and specific (what it does, not hype).
5. Open a pull request.

## Quality bar

- **Real links only.** Prefer the arXiv `abs` page; add `code` / `project` links when they exist.
- **Verify before you add.** Confirm the arXiv ID resolves to the paper you cite.
- **No duplicates.** Search the list first; if a paper fits two sections, place it once in the best-fit section and cross-reference.
- **Keep it on-topic.** This list is about *diffusion models for language / discrete-token sequences* (text, code, and closely related "language of X" sequences like proteins). General image diffusion belongs elsewhere unless it directly enables text/unified generation.
- **Chronological within a section** (oldest → newest) unless a section says otherwise.

## Reporting problems

Open an issue for broken links, wrong attributions, or miscategorized entries. Corrections are as valuable as additions.

## Order-of-merge

Maintainers may reorder, merge, or reword entries for consistency. By contributing you agree to release your contribution under the repository's [CC0](LICENSE) dedication.
